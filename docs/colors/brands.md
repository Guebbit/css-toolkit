# Brand colors

Brand palette exported from `src/colors/brands.js`.

Includes solid colors like:

- `brandFacebook`
- `brandYoutube`
- `brandInstagram`
- `brandTiktok`

And gradients:

- `gradientBrandInstagram`
- `gradientBrandTiktok`

Example:

```js
import { brandFacebook, gradientBrandInstagram } from '@guebbit/css-toolkit/src/colors/brands.js';
```
