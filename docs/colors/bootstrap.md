# Bootstrap colors

Palette exported from `src/colors/bootstrap.js`.

Includes:

- base colors (`blue`, `indigo`, `purple`, …, `black`, `white`)
- grayscale (`gray100..gray900`)
- generated shades (`<color>100..900`) via tint/shade helpers

Use this module when you need direct color constants in JavaScript:

```js
import { blue500, gray900 } from '@guebbit/css-toolkit/src/colors/bootstrap.js';
```
